整合了之前所寫的圖形跟聲音(bmp.wav)，目前有三種資料隱藏方法：PVD、PVD-LSB、PVD-matrix_embbeding(演算法可以看我的presentation)，環境是在Ubuntu上。 若想要compile下列檔案請參考 README 。

  • steganography.cpp : 主要是main function，接收使用者所下的指令參數，做相對應的隱藏。
  • format.cpp format.h : 負責讀寫檔案，支援BMP跟WAV兩種格式。
  • datahiding.cpp datahiding.h : 裡面共有三種隱藏方式，目前能夠將檔案或字串藏進BMP或WAV。

  • psnr.cpp : 針對我做的方法來計算psnr值。

    以上程式也能在我的 GitHub 上下載。
