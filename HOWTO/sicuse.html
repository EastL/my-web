How to use SIC

我的linux是Ubuntu，iso檔可以直接在官網下載，我的版本是13.04，以下說明都是在 ubuntu環境上執行。

在學習如何使用SIC simulator之前，首先要先具備一些基本工具：pascal compiler, SIC source

  • pascal compiler可以下這道指令安裝： sudo apt-get install fp-compiler-2.6.0
  • SIC source 可以到ftp rohan.sdsu.edu.tw (faculty/beck/SystemSoftware.tar這檔案)下 載 (課本的VII頁數)

COMPILE

這邊有一個device為SRCFILE，他是放你寫好的程式碼，為input device
下載完把tar檔打開後，裡面有一個sicasm.pas檔，用pascal compiler compile這個檔案 指令：fpc sicasm.pas

我之後要測的程是在別的資料夾，在這裡我先把東西都丟到我將來要操作的資料夾裡 (test.asm是我自己寫好的程式碼)

sicasm就變成可執行檔，當你要compile SIC程式碼時，先把你寫好的程式複製到SRCFILE裡 面

再執行sicasm，後面接的參數是你寫好的程式碼檔案名稱。


執行完後會出現OBJFILE，LISFILE，INTFILE這3個檔案

使用simulator

使用之前，必須先知道一些device的功用，DEV00、DEVF1、DEVF2、DEVF3為input device， DEV04、DEV05、DEV06、LOG為output device

DEV00放入sicboot.obj
DEVF1放入sicldr.obj
DEVF2放入你自己寫完編譯好的obj檔
DEVF3是你要輸入到你的程式裡面的input file(程式沒有要輸入資料的話不用複製)

接下來只要再將sicsim.pas編譯程可執行檔

就能夠使用simulator了
